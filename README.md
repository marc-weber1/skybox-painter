# skybox-painter

To render for visual studio, just use:
cmake -Bbuild
and use the build folder as a project.

!! THIS REPOSITORY HAS SUBMODULES
When cloning, use:
git clone (url) -recursive
Or:
git clone (url)
git submodule init
git submodule update

Possible extensions:
https://github.com/eug/awesome-opengl#libraries
